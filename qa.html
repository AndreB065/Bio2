Q&A with Andre

What is your first memory of using a computer?
- My memory is not too good and I can't remember the sequence of things that happened so long ago, but one of my earliest memories of using a computer is messing around with an old IBM word processor my mom had.

What was the first computer you bought and how powerful was it?
- The first computer I ever bought for myself is the desktop that I currently use to this day, an Acer AM5100-B5404A, with an AMD Athlon 64X2 processor and 3GB of RAM. It was pretty good for 2008.

What was the first programming language you learned?
- The first programming language I learned was C++.

How did it come about that it became the first programming language you learned?
- It's the first one I sat in a class to learn because it was what was suggested by my school advisor at the time.

What languages have you programmed in? Which was the easiest? Why?
- I've programmed with C++, C#, Java, JavaScript, PHP, Ruby, and Python. My memory is not too good, as I've mentioned, so I can't really say which was easiest. The one that's in my mind now is the last one I used, JavaScript. So, I would default to that being the easiest at the moment. In a couple months, I'll probably be saying it's PHP.

What is your favorite programming language? Why?
- I can't say that I have a favorite programming language because I tend to fully immerse myself in the language I'm using at the time of the project. It's almost like a new experience every time I switch to a different programming language.
